# 🔍 FilePathFinder

`FilePathFinder` は、膨大なフォルダー構造を瞬時にスキャンしてリスト化し、後から必要なファイルを**爆速で検索・パスコピー**できる超軽量のWindows GUIツールです。

「あのファイル、サーバーや外付けHDDのどこに置いたっけ…？」
「ネットワークドライブの検索が遅すぎて仕事にならない…」

そんなストレスから解放され、手元でローカルファイルを扱うかのような快適さで目的のファイルパスへとアクセスできます。

---

## ✨ 主な特徴とメリット

* **爆速のインデックス検索**
  一度フォルダーをスキャンしてリスト（`.gz` 圧縮ファイル）を作っておけば、数万・数十万のファイルからでも一瞬でキーワード検索が行えます。
* **スペース区切りのAND検索対応**
  「2026 提案書 pdf」のようにスペースで区切ることで、目的のファイルを一瞬で絞り込めます。
* **スマートな検索結果（HTML出力）**
  検索結果は自動的にモダンなWebページ（HTML）として開かれます。ライトモード/ダークモードの両方に対応し、スクロールバーまで心地よく馴染むデザインです。
* **ファイル名に集中できる「フォルダ名非表示」スイッチ**
  画面上のスイッチでフォルダ経路をサッと隠し、ファイル名だけをスッキリ一覧表示できます。「📁 フォルダ」ボタンにマウスカーソルを乗せたときだけ、その行のフォルダ名が浮かび上がる親切設計です。
* **ワンクリックでパスをコピー**
  * **行をクリック**: フルパスをクリップボードにコピー
  * **「📁 フォルダ」ボタンをクリック**: そのファイルが属する「親フォルダのパス」をコピー（そのままエクスプローラーのアドレスバーに貼り付けてジャンプできます）

---

## 🚀 使い方

### 1. 準備（リストの作成）
1. `FilePathFinder.exe` を起動します。
2. 右上の **[新しいリストを作る]** をクリックし、スキャンしたいフォルダー（ローカル、外付けHDD、ネットワークドライブなど）を選択します。
3. スキャンが完了すると、ツールと同じ場所に `filelist.gz` が生成されます。
4. 💡 **便利技**: 生成された `filelist.gz` の名前を `Cドライブ.gz` や `共有サーバー.gz` のようにリネームしておくと、ツール上のドロップダウンで簡単に切り替えられるようになります。

### 2. 検索とパスの取得
1. 検索対象のファイルリスト（ドロップダウン）を選択します。
2. キーワードを入力して【Enter】または **[検索]** をクリックします。
3. ブラウザで検索結果の画面が開きます。
4. **結果画面での操作**:
   * さらにキーワードを入力すれば、その場で**リアルタイムに自動絞り込み**が行われます（スペースでAND検索も可能）。
   * 画面をスペースキーで快適にスクロールできます（入力欄へ勝手にフォーカスが奪われません）。
   * 必要な行をクリックしてフルパスを取得するか、右端に出現する「フォルダ」ボタンで親パスを取得してください。

## 🛠️ 動作環境
* **OS**: Windows 7 / 8 / 10 / 11
* **要件**: .NET Framework 4.5 以上（特別な環境構築なしで、そのまま実行可能です）

---

## 📝 開発者向けノート / 仕様
* スキャンデータは `.gz` 形式で高度に圧縮して保存されるため、ディスク容量をほとんど圧迫しません。
* 検索結果のHTMLは、スタンドアロン（単体）で動作するクリーンなJavaScript/CSSで構成されており、外部への通信などは一切行わないためセキュリティ的にも安全です。
