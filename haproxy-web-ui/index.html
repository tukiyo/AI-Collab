システム・設定ファイルのバックアップ (ダウンロード)

現在の設定ファイルや、この管理コンソールのプログラム本体をダウンロードしてバックアップします。

ロードバランサー設定

⬇ HAProxy /etc/haproxy/haproxy.cfg
⬇ Keepalived /etc/keepalived/keepalived.conf

OS・ネットワーク設定

⬇ Netplan /etc/netplan/01-netcfg.yaml
⬇ Sysctl /etc/sysctl.conf
⬇ iptables /etc/iptables/rules.v4

管理コンソールプログラム

⬇ Pythonバックエンド /opt/lb_admin/lb_admin.py
⬇ Webフロントエンド /opt/lb_admin/index.html

HAProxy設定のリストア

お手元にある HAProxy の設定ファイル（haproxy.cfg）を選択して、本番環境にリストア（復元）します。
※Pythonやネットワーク設定等のリストアは、動作不具合を防ぐためサーバー上で直接上書きを行ってください。
